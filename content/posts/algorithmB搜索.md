# 一. 图的存储

[B02 图的存储——信息学奥赛算法_哔哩哔哩_bilibili](https://www.bilibili.com/video/BV1UT411V7u4/?spm_id_from=333.337.search-card.all.click&vd_source=86d06d10a03dcd75fe3a632194bc3c3c)

## 1. 链式前向星

**链式前向星** 数据结构来构建和存储树（本质是无向图），核心逻辑围绕**邻接表**展开，将每个边的编号作为边的唯一标识，然后将这个标识映射到一个边数组上借此来查询边上的终点，以及同一个起点的上一条边

就是采用三个数组来构建：

1. h[N]:h[x]表示节点x的新的边的编号
2. to[2*N]:to[i]存储编号为i的边的终点
3. ne[2*N]:to[j]存储与编号为j的边相同起点的边的编号，也就是原来的h[a];



### . 核心数据结构定义



```c++
int h[N];    // 表头数组，h[x] 存储以节点 x 为起点的第一条边的编号
int to[2*N]; // 存储每条边的目标节点（终点）
int ne[2*N]; // 存储同一起点的下一条边的编号（链表指针）
int tot;     // 边的计数器（从1开始编号，避免与初始0冲突）
```

树是无向图，每条边需要存储两次（`a->b` 和 `b->a`），因此数组大小开为 `2*N`。

### 2. 加边函数 `add`



```c++
void add(int a,int b){ 
  to[++tot] = b;     // 新边的终点是 b
  ne[tot] = h[a];    // 新边的下一条边指向原 h[a] 对应的边
  h[a] = tot;        // 更新 h[a] 为新边的编号，成为新的表头
}
```

这是链式前向星的核心操作，本质是**头插法**向节点 `a` 的邻接表中添加一条指向 `b` 的边。

### 3. 树的构建（主函数）



```c++
scanf("%d",&n); 
for(int i=1,a,b; i<n; i++){
  scanf("%d%d",&a,&b);
  add(a,b);  // 添加 a 到 b 的边
  add(b,a);  // 添加 b 到 a 的边（无向树必须双向存储）
}
```

- 读入节点总数 `n`，树有 `n-1` 条边。
- 对每条输入的边 `(a,b)`，**双向调用 `add`**，确保从 `a` 能找到 `b`，从 `b` 也能找到 `a`。

### 4. 树的遍历（DFS）

后续的 `dfs(1,0)` 基于构建好的邻接表遍历树：



```c++
void dfs(int x,int f){ 
  for(int i=h[x];i;i=ne[i]){  // 遍历 x 的所有邻边
    int y=to[i];              // 邻边的终点 y
    if(y!=f){                 // 避免回走父节点（f 是 x 的父节点）
      // 预处理深度、祖先、前缀和数组...
      dfs(y,x);                // 递归遍历子节点
    }
  }
}
```

通过 `h[x]` 遍历节点 `x` 的所有邻居，再通过 `ne[i]` 切换到下一条边，直到 `ne[i]==0` 表示该节点的邻边遍历完毕。

### 总结

代码通过**链式前向星**将无向边双向存储为邻接表，再通过 DFS 递归遍历邻接表完成树的构建与信息预处理（深度、祖先、前缀和等），这是处理大规模树 / 图问题（n≤3×105）的高效标准写法。
