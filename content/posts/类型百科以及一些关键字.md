## 一、C++ 常见头文件及作用

### 1. 输入输出相关



```c++
#include <iostream>   // 核心：cin, cout, cerr, endl
#include <iomanip>    // 格式化输出：setw, setprecision, fixed
#include <fstream>    // 文件读写：ifstream, ofstream, fstream
```

### 2. 字符串



```c++
#include <string>     // std::string 字符串类型
#include <cstring>    // C 风格字符串：strlen, strcpy, strcmp
```



### 3 算法 / 工具

```c++
#include <algorithm>  // 排序、查找：sort, find, reverse, swap
#include <cmath>      // 数学函数：sqrt, pow, abs, sin, cos
#include <cstdlib>    // 随机数：rand, srand；系统函数：system
#include <ctime>      // 时间：time(0)（用于生成随机种子）
#include <cstdio>     // C 语言输入输出：printf, scanf
```

### 4. 其他常用

```c++
#include <cctype>     // 字符判断：isdigit, isalpha, tolower
#include <sstream>    // 字符串流：字符串转数字、数字转字符串
#include <cstddef> 	  //用于定义size_t以及difference_type
```





# 二、常见关键字

## 1.变量存储相关

`static`：

- 局部变量：延长生命周期，程序结束才销毁
- 类成员：属于类，不属于对象

`extern`：声明外部变量 / 函数



## 2.函数调用相关

`inline`：内联函数（减少函数调用开销）

`constexpr`：编译期常量（C++11）

`noexcept`:表示函数不会发生异常，用于一些底层不应该发生异常的操作

## 3.面向对象相关

`virtual`：虚函数（实现多态）

`override`：明确重写虚函数（C++11）

`final`：禁止继承 / 重写

`namespace`：命名空间（避免命名冲突）
